Git Setup

Global Git Configuration

Configure Git account before cloning project. 

# Set username
git config --global user.name "Your Name"

# Set email
git config --global user.email "your_email@example.com"

# Verify setup
git config --global --list
Clone Repository

Clone Project

Download source code from GitLab server. 

# Go to desktop
cd Desktop

# Clone project
git clone http://YOUR_GITLAB_SERVER/acleda-mobile-ios.git

# Open folder
cd acleda-mobile-ios
CocoaPods

Install Pods

Install required iOS libraries. 

# Install pods
pod install

If CocoaPods missing: 

# Install CocoaPods
sudo gem install cocoapods
Xcode

Open Workspace

Open workspace for CocoaPods support. 

# Open workspace
open YourProject.xcworkspace
SPM

Resolve Packages

Download Swift Package dependencies. 

File
→ Packages
→ Resolve Package Versions
Branch

Create Feature Branch

Create new working branch before coding. 

# Switch develop
git checkout develop

# Pull latest code
git pull origin develop

# Create branch
git checkout -b feature/home-ui
Status

Check Changes

Show changed and untracked files. 

# Check status
git status
Git Add

Add Files

Add files before commit. 

# Add all files
git add .

# Add single file
git add Sources/HomeViewController.swift
Commit

Commit Changes

Save changes locally. 

# Commit code
git commit -m "feat: improve login validation"
Push

Push Code

Upload code to GitLab server. 

# First push
git push -u origin feature/home-ui

# Next push
git push
Merge Request

Create MR

Submit code for review and merge. 

1. Open GitLab
2. Go to Merge Requests
3. Click New MR
4. Select source branch
5. Select target branch
6. Create Merge Request
Workflow

Daily Development Flow

Recommended daily Git workflow. 

# Update develop
git checkout develop
git pull origin develop

# Return to branch
git checkout your-branch

# Sync latest develop
git merge develop
Best Practice

Development Rules

  • Never commit directly to main branch
  • Always use feature branches
  • Pull latest develop before coding
  • Review files before pushing
  • Use meaningful commit messages
  • Create MR for every update

Troubleshooting

Fix Common Issues

Resolve CocoaPods and Xcode issues. 

# Update pod repo
pod repo update

# Install pods again
pod install

# Remove derived data
rm -rf ~/Library/Developer/Xcode/DerivedData
