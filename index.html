iOS Developer Guide

Git Local Server iOS

Complete Git workflow & local server setup for iOS developers.

Git Setup

Global Git Configuration

Configure Git account before cloning project. 

# Set username
git config --global user.name "Your Name"

# Set email
git config --global user.email "your_email@example.com"

# Verify setup
git config --global --list
Clone Repository

Clone Project

Download source code from GitLab server. 

# Go to desktop
cd Desktop

# Clone repository
git clone http://YOUR_GITLAB_SERVER/repo-name.git

# Open project folder
cd repo-name
Branch

Create Feature Branch

Create new branch before coding. 

# Switch develop branch
git checkout develop

# Pull latest code
git pull origin develop

# Create new feature branch
git checkout -b feature/home-ui
Status

Check Git Status

Verify changed and staged files. 

# Show current changes
git status

# Short status
git status -s
Stage

Add Files

Add source files before commit. 

# Add all files
git add .

# Add specific file
git add HomeViewController.swift
Commit

Commit Changes

Save changes locally. 

# Commit code
git commit -m "feat: improve login validation"

# Amend last commit
git commit --amend
Push

Push Code

Upload code to GitLab server. 

# First push
git push -u origin feature/home-ui

# Next push
git push
Pull

Pull Latest Code

Sync latest changes from remote branch. 

# Pull develop branch
git pull origin develop

# Fetch only
git fetch
History

View Commit History

Show commit records and logs. 

# Full log
git log

# One line log
git log --oneline

# Graph log
git log --oneline --graph
Branches

Manage Branches

View and delete branches. 

# Show branches
git branch

# Show all branches
git branch -a

# Delete branch
git branch -d feature/home-ui

# Force delete branch
git branch -D feature/home-ui
Stash

Temporary Save

Save unfinished changes temporarily. 

# Save changes
git stash

# View stash
git stash list

# Restore stash
git stash pop
Compare

View Differences

Compare modified code changes. 

# Show difference
git diff

# Show staged difference
git diff --cached
Reset

Reset Changes

Undo local changes safely. 

# Reset staged files
git reset

# Hard reset
git reset --hard

# Reset commit
git reset --hard COMMIT_ID
Clean

Remove Untracked Files

Delete temporary and unused files. 

# Remove files
git clean -fd
Rebase

Rebase Branch

Keep branch updated cleanly. 

# Rebase develop
git rebase develop

# Continue rebase
git rebase --continue

# Abort rebase
git rebase --abort
Merge

Merge Branch

Merge feature branch into current branch. 

# Merge branch
git merge feature/home-ui

# Abort merge
git merge --abort
CocoaPods

Install Pods

Install required iOS libraries. 

# Install pods
pod install

# Update pods
pod update
Xcode

Open Workspace

Open project workspace. 

# Open workspace
open YourProject.xcworkspace
Merge Request

Create Merge Request

Submit code for review. 

1. Open GitLab
2. Go to Merge Requests
3. Click New MR
4. Select source branch
5. Select target branch
6. Create Merge Request
