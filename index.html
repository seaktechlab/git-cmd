iOS Developer Professional Guide

Git Workflow & Local Server Setup

Complete Git, GitLab, CocoaPods, GitFlow and iOS project setup guide for professional development teams.

Git Setup

Global Git Configuration

Configure Git account before starting project. 

# Set username
git config --global user.name "Your Name"

# Set email
git config --global user.email "your_email@example.com"

# Verify setup
git config --global --list
SSH

SSH Setup GitLab

Generate SSH key for secure GitLab access. 

# Generate SSH key
ssh-keygen -t ed25519 -C "your_email@example.com"

# Start SSH agent
eval "$(ssh-agent -s)"

# Add SSH key
ssh-add ~/.ssh/id_ed25519

# Show public key
cat ~/.ssh/id_ed25519.pub
Clone

Clone Repository

Download project from GitLab server. 

# Go desktop
cd Desktop

# Clone project
git clone git@gitlab.com:project/repo.git

# Open folder
cd repo-name
Git Ignore

Create .gitignore

Ignore unnecessary iOS files. 

# Create gitignore
touch .gitignore

CocoaPods

Install Pods

Install required libraries. 

# Install pods
pod install

# Update pods
pod update
Xcode

Open Workspace

Open workspace instead of xcodeproj. 

open YourProject.xcworkspace
Branch

Create Feature Branch

Start feature development. 

# Switch develop
git checkout develop

# Pull latest
git pull origin develop

# Create feature branch
git checkout -b feature/home-ui
Status

Check Status

Check modified files. 

# Show status
git status

# Short status
git status -s
Stage

Add Files

Stage source code before commit. 

# Add all
git add .

# Add specific file
git add HomeVC.swift
Commit

Commit Changes

Save local source changes. 

# Commit
git commit -m "feat: improve login validation"

# Amend commit
git commit --amend
Push

Push Code

Upload source code to GitLab. 

# First push
git push -u origin feature/home-ui

# Next push
git push
Force Push

Force Push

Overwrite remote branch history. 

git push --force
⚠️ Use carefully. Can overwrite remote history.
Pull

Pull Latest Code

Sync latest source from remote. 

# Pull branch
git pull origin develop

# Fetch only
git fetch
History

Commit History

View Git commit history. 

# Full history
git log

# One line
git log --oneline

# Graph
git log --oneline --graph
Stash

Temporary Save

Save unfinished work temporarily. 

# Save
git stash

# View stash
git stash list

# Restore stash
git stash pop
Diff

View Difference

Compare changed source code. 

# Difference
git diff

# Staged difference
git diff --cached
Reset

Undo Changes

Reset local changes safely. 

# Soft reset
git reset --soft HEAD~1

# Hard reset
git reset --hard HEAD~1
Revert

Revert Commit

Safely undo shared commit. 

git revert COMMIT_ID
Cherry Pick

Apply Specific Commit

Copy commit into current branch. 

git cherry-pick COMMIT_ID
Rebase

Rebase Branch

Keep branch history clean. 

# Rebase develop
git rebase develop

# Continue
git rebase --continue

# Abort
git rebase --abort
Merge

Merge Branch

Merge feature branch. 

# Merge branch
git merge feature/home-ui

# Abort merge
git merge --abort
Conflict

Resolve Conflict

Fix merge conflict manually. 

# Check conflict
git status

# Add resolved files
git add .

# Continue merge
git commit
Release

Tag Release

Create production release tag. 

# Create tag
git tag v1.0.0

# Push tags
git push --tags
Remote

Remote URL

Manage Git remote repository. 

# Show remote
git remote -v

# Change remote URL
git remote set-url origin NEW_URL
GitFlow

Git Branch Strategy

Professional Git workflow structure. 

main       -> production
develop    -> staging
feature/*  -> new features
hotfix/*   -> urgent fixes
release/*  -> release preparation
